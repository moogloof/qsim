# Quantum Sim
Python and C versions.

## Requirements
- Python3
- venv

## Running
Setup the virtual environment:
```sh
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
Run program in the python directory:
```sh
python python/sim.py
```
